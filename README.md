💖 Our Love Story – Streamlit App

A beautiful and interactive Streamlit web application created to express love through a personalized story, images, and emotions. This project is designed to showcase a romantic journey, especially meaningful for long-distance relationships.

🌟 Features
💞 Personalized love story sections
📸 Display of your real images
🎨 Clean and beautiful UI with custom styling
🌍 Section on Long Distance Relationship (Pros & Cons)
💌 Interactive “I Love You” reveal button
❤️ Emotional and memorable user experience

⚙️ Requirements

Make sure you have Python installed (3.8+ recommended)

Install required libraries:

pip install streamlit pillow
🚀 How to Run the App
Open terminal / command prompt
Navigate to your project folder:
cd your_project_folder
Run the Streamlit app:
streamlit run app.py
Your app will open in a browser automatically 🎉
